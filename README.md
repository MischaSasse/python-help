# python-help
a repo for students who just started as a way to understand the basics

This repo will include the following concepts:
- 1.0 variables and datatypes
- 2.0 if statements
- 3.0 lists / dictionaries
- 4.0 loops (for, while and foreach)
- 5.0 functions
- 6.0 SQL (niet met python)
- 7.0 objects
- 8.0 classes + relations (inheritance)
- 8.0 private and public variables
- 9.0 basic OOP
